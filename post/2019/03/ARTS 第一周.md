# Algorithm

[238. Product of Array Except Self](https://leetcode.com/problems/product-of-array-except-self)：You are given coins of different denominations and a total amount of money *amount*. Write a function to compute the fewest number of coins that you need to make up that amount. If that amount of money cannot be made up by any combination of the coins, return `-1`.
